Displays a user role next to the comment from a user registered on your site with the ability to customize a background color.

== Description ==

Adds a comment author label next to comments and allows you to customize the background color of the label for roles registered on your site.

== Installation ==

1. Upload `comment-author-role-label` unzipped file to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Select the background colors you want for each role on your site and save your changes.

== Frequently Asked Questions ==

= Where is this added in the admin? =

Under the Comments section labeled 'Author Labels'.
